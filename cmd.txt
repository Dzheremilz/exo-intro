man : manuel du systeme
option -h ou --help
!! : relance la dernière commande

ls : liste le répertoire courant
  -f ne trie pas le répertoire, active les options -a et -U
  -i numméro d'index de chaque fichier
  -l sous forme de longue liste
  -a all y compris les fichiers commençant par . (caché)
  -h human-readable ajoute l'unité de taille par une lettre
cd : change directory, permet de se deplacer dans les répertoires
pwd : print working directory, affiche le chemin d'accès vers le répertoire où on se situe
mkdir : make directory, crée un répertoire
nvm
node
rm : remove, supprime fichier et répertoire
mv : move, déplacer ou renommer fichier et répertoire 
cp : copy, copie fichier et répertoire
apt : Advanced Packaging Tool, gestion de paquets, recherche et installation
history : historique des commandes




sudo : Substitute User DO
sudo permet d'utiliser une commande en tant que superutilisateur (root) ou tout autre utilisateur, il faudra l'utilise pour toute action lié au logiciel : installation/desinstallation/mise à jour
